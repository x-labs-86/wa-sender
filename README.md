### Requirement To Install

- NodeJS v16.20.0 (LTS)
- NPM v8.19.4
- Nativescript v8.5.x

### How to run?

First install all dependecies in `package.json` with run this command

```bash
npm install
```

#### andorid

```bash
ns run android
```

#### ios

```bash
ns run ios
```

### How to build?

First install all dependecies in `package.json` with run this command

```bash
npm install
```

#### andorid

```bash
ns build android
```

#### ios

```bash
ns build ios
```
